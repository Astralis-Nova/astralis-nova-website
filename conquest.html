FIELD REPORT

What Makes It Different

Conquest is publicly listed as a custom PvE server featuring live events, deep crafting, active development, and a PK system. Because the world contains custom content, retail-era guides may not perfectly match Conquest quests, balance, rewards, or progression.

Use the Conquest Discord as the final authority for current patches, custom quests, event schedules, server rules, downloads, and progression advice.
